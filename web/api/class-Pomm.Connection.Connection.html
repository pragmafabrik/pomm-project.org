Class Connection

Pomm\Connection\Connection

Manage a connection to the database. Connection is a pool of Map and PreparedQuery instances. It holds the IdentityMapper. Connection also proposes handy methods to deal with transactions.

Pomm\Connection\Connection implements Psr\Log\LoggerAwareInterface
