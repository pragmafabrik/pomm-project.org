19:  * NOTE: Here is the coding convention for value conversion TO postgres:
20:  * Values are always surrounded by simple quotes with the type prefixed.
21:  * Example: int4 '8', bool 'true', varchar 'a b c d'
22:  *
23:  * Null values must be type cast inline and be uppercase.
24:  * Example: NULL::int4, NULL::timestamp, NULL::circle
25:  *
26:  * Arrays are just declared with 'array[…]' and type cast inline.
27:  * Example: array[int4 '8',int4 '12']::int4[]
28:  *
29:  * Complex types with an existing constructor must use it.
30:  * Exemple: circle(point(1.23,2.34), 5.67), hstore('"a" => "b"')
31:  *
32:  * @package Foundation
33:  * @copyright 2014 Grégoire HUBERT
