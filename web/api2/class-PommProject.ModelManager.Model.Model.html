# createDefaultProjection( )

createDefaultProjection

createDefaultProjection

This method creates a projection based on the structure definition of the underlying relation. It may be used to shunt parent createProjection call in inherited classes. This method can be used where a projection that sticks to table definition is needed like recursive CTEs. For normal projections, use createProjection instead.
